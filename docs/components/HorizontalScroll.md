#### 横向滚动

author: 陈书航

![img](./img/horizontalScrollImg.png)

#### 组件路径

`@/components/ecpp/HorizontalScroll`

#### API

| 参数     | 说明                                     | 类型    | 默认值 | 是否必填 |
| -------- | ---------------------------------------- | ------- | ------ | -------- |
| id       | 需要滚动的 id                            | string  | -      | 是       |
| showMove | 是否需要滚动，可根据列表数据长度进行判断 | boolean | false  | 是       |

#### slot

| 参数    | 说明               |
| ------- | ------------------ |
| content | 需要滚动的组件内容 |
